Description of *panstarrs_photometry.fits files.

####### *panstarrs_photometry.fits #######

Columns 1-37            - Default SoFiA catalogue columns

SEG_X	 	        - PanSTARRS segment x position [pixels]
SEG_Y	 	        - PanSTARRS segment y position [pixels]
SEG_BA	 	        - PanSTARRS segment axis ratio b/a
SEG_PA	 	        - PanSTARRS segment position angle [degrees]
RADIUS_R_ISO25	 	- PanSTARRS r-band 25 mag / arcsec^2 isophotal radius [arcsec]
RADIUS_R_ISO26	 	- PanSTARRS r-band 26 mag / arcsec^2 isophotal radius [arcsec]
RADIUS_R_ISO27	 	- PanSTARRS r-band 27 mag / arcsec^2 isophotal radius [arcsec]
RADIUS_R_50	 	- PanSTARRS r-band effect (r50) radius [arcsec]
RADIUS_R_90	 	- PanSTARRS r-band r90 radius [arcsec]
R_mag25	 	        - Total r-band magnitude measured w/in 
			  25 mag / arcsec^2 isophotal radius aperture [mag]
R_mag26	 	        - Total r-band magnitude measured w/in 
			  26 mag / arcsec^2 isophotal radius aperture [mag]
R_mag27	 		- Total r-band magnitude measured w/in 
			  27 mag / arcsec^2 isophotal radius aperture [mag]
G_mag25	 	        - Total g-band magnitude measured w/in 
			  25 mag / arcsec^2 isophotal radius aperture [mag]
G_mag26	 	        - Total g-band magnitude measured w/in 
			  26 mag / arcsec^2 isophotal radius aperture [mag]
G_mag27	 		- Total g-band magnitude measured w/in 
			  27 mag / arcsec^2 isophotal radius aperture [mag]
