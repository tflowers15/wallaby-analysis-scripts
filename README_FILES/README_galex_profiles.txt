Description of GALEX NUV-band *profiles.fits files, which are located in 
'basedir + /MULTIWAVELENGTH/GALEX/PROFILES/', where * is the galaxy name.

####### *profiles.fits #######

RADIUS_ANNULUS			- Annulus radius (outer edge) [pixel]
AREA_ANNULUS 	        	- Annulus area [pixel^2]
ADU_ANNULUS_NUV	 		- Total annulus flux (background subtracted) [ADU]
MAG_ANNULUS_NUV	 		- Annulus magnitude (background subtracted) [mag]
ADU_ANNULUS_BK_NUV		- Total annulus flux (includes background) [ADU]
MAG_ANNULUS_BK_NUV 		- Annulus magnitude (includes background) [mag]

RADIUS_APERTURE			- Aperture radius [pixel]
AREA_APERTURE 	        	- Aperture area [pixel^2]
ADU_APERTURE_NUV	 	- Total aperture flux (background subtracted) [ADU]
ADU_APERTURE_ERR_NUV		- Error in aperture flux (background subtracted) [ADU]
ADU_DIFF_APERTURE_NUV   	- Difference in adjacent aperture fluxes 
			  	  (background subtracted) [ADU]
MAG_APERTURE_NUV 		- Aperture magnitude (background subtracted) [mag]
MAG_DIFF_APERTURE_NUV 		- Difference in adjacent aperture magnitudes 
			  	  (background subtracted) [mag]

ADU_APERTURE_BKGD_NUV		- Total aperture flux (includes background) [ADU]
ADU_DIFF_APERTURE_BKGD_NUV   	- Difference in adjacent aperture fluxes 
			  	  (includes background) [ADU]
MAG_APERTURE_BKGD_NUV 		- Aperture magnitude (includes background) [mag]
MAG_DIFF_APERTURE_BKGD_NUV 	- Difference in adjacent aperture magnitudes 
			  	  (includes background) [mag]
