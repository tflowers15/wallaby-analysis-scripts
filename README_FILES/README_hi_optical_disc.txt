Description of *hi_optical_disc.fits files.

####### *hi_optical_disc.fits #######

Columns 1-37            	- Default SoFiA catalogue columns

FLUX_INNER_PS25	 	        - HI flux w/in aperture defined by r-band 
				  25 mag / arcsec^2 isophotal radius (flux
				  corrected) [Jy Hz / beam]
SURFACE_DENSITY_INNER_PS25	- Average HI surface density w/in aperture r-band 
				  25 mag / arcsec^2 isophotal radius (flux
				  corrected) [Msol / pc^2]

FLUX_INNER_PS50	 	        - HI flux w/in aperture defined by r-band effective 
				  radius (flux corrected) [Jy Hz / beam]
SURFACE_DENSITY_INNER_PS50	- Average HI surface density w/in effective 
				  radius (flux corrected) [Msol / pc^2]

FLUX_INNER_PS90	 	        - HI flux w/in aperture defined by r-band r90 
				  radius (flux corrected) [Jy Hz / beam]
SURFACE_DENSITY_INNER_PS90	- Average HI surface density w/in r90 
				  radius (flux corrected) [Msol / pc^2]
