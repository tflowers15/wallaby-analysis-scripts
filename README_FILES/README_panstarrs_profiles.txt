Description of PanSTARRS g- and r-band *profiles.fits files, which are located in 
'basedir + /MULTIWAVELENGTH/PANSTARRS/PROFILES_BKGDSUB/', where * is the galaxy name.

####### *profiles.fits #######

+++++++ r-band +++++++

RADIUS_ANNULUS_R	- Annulus radius (outer edge) [pixel]
AREA_ANNULUS_R 	        - Annulus area [pixel^2]
ADU_ANNULUS_R	 	- Total annulus flux [ADU]
ADU_ANNULU_ERR_R 	- Error in annulus flux [ADU]
MAG_ANNULUS_R	 	- Annulus magnitude [mag]

RADIUS_APERTURE_R 	- Aperture radius [pixel]
AREA_APERTURE_R	 	- Aperture area [pixel^2]
ADU_APERTURE_R 	        - Total aperture flux [ADU]
ADU_APERTURE_ERR_R	- Error in aperture flux [ADU]
MAG_APERTURE_R 	        - Aperture magnitude [mag]

EXPTIME_R	 	- Image exposure time (required to convert ADU to mag) [s]
SKY_MAG_R 	        - Local sky background magnitude [mag]

ADU_ANNULU_ERR_G 	- Error in annulus flux [ADU]
MAG_ANNULUS_G	 	- Annulus magnitude [mag]

RADIUS_APERTURE_G 	- Aperture radius [pixel]
AREA_APERTURE_G	 	- Aperture area [pixel^2]
ADU_APERTURE_G 	        - Total aperture flux [ADU]
ADU_APERTURE_ERR_G	- Error in aperture flux [ADU]
MAG_APERTURE_G 	        - Aperture magnitude [mag]

SKY_MAG_G 	        - Local sky background magnitude [mag]
