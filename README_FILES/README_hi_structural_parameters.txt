Description of *hi_structural_parameters.fits files.

####### *hi_structural_parameters.fits #######

Columns 1-37            	- Default SoFiA catalogue columns

OBJECTS	 	        	- Galaxy name

RADIUS_ISO	 	        - Isodensity HI radius [arcsec]
SURFACE_DENSITY_ISO	 	- Average HI surface density w/in isodensity 
				  radius [Msol / pc^2]
RADIUS_ISO_ERR	 	        - Uncertainty in HI radius [arcsec]

RADIUS_EFF	 	        - Effective HI radius [arcsec]
SURFACE_DENSITY_EFF	 	- Average HI surface density w/in effective 
				  radius [Msol / pc^2]

DIAMETER_MAJOR	 	        - Isodensity HI diameter (major axis) [arcsec]
DIAMETER_MINOR	 	        - Isodensity HI minor axis [arcsec]
AXIS_RATIO_BA	 	        - HI axis ratio (b/a; GAUSSIAN_MIN/GAUSSIAN_MAJ)

TOTAL_FLUX	 	        - Total HI flux measured from moment 0 map [Jy Hz]

RADIUS_ISO_SCALE	 	- Isodensity HI radius (flux corrected) [arcsec]
SURFACE_DENSITY_ISO_SCALE	- Average HI surface density w/in isodensity 
				  radius (flux corrected) [Msol / pc^2]
RADIUS_ISO_ERR_SCALE	 	- Uncertainty in HI radius (flux corrected) [arcsec]

RADIUS_EFF_SCALE	 	- Effective HI radius (flux corrected) [arcsec]
SURFACE_DENSITY_EFF_SCALE	- Average HI surface density w/in effective 
				  radius (flux corrected) [Msol / pc^2]

DIAMETER_MAJOR_SCALE	 	- Isodensity HI diameter (flux corrected) [arcsec]

TOTAL_FLUX_SCALE	 	- Total HI flux measured from moment 0 map
 				  (flux corrected) [Jy Hz]

GAUSSIAN_RA	 	        - 2D Gaussian fit to moment 0 map RA
GAUSSIAN_DEC	 	        - 2D Gaussian fit to moment 0 map Dec
GAUSSIAN_MAJ	 	        - 2D Gaussian fit to moment 0 map major axis
GAUSSIAN_MIN	 	        - 2D Gaussian fit to moment 0 map minor axis
GAUSSIAN_PA	 	        - 2D Gaussian fit to moment 0 map position angle
