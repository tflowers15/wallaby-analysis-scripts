Description of WALLABY HI *profiles.fits files, which are located in 
'basedir + /HI_DERIVED_PRODUCTS/PROFILES/', where * is the galaxy name.

####### *profiles.fits #######

RADIUS_ANNULUS	 		- Elliptical annulus radii [pixels]
AREA_ANNULUS 	        	- Annulus area [pixels^2]
FLUX_ANNULUS	 		- Total flux w/in annulus [Jy Hz]
ERROR_ANNULUS 	        	- Uncertainty in flux w/in annulus [Jy Hz]
SURFACE_DENSITY_ANNULUS	 	- Average HI surface density w/in annulus (produces 
			  	  surface density profile) [Msol / pc^2]

RADIUS_APERTURE 		- Elliptical aperture radii [pixels]
AREA_APERTURE	 		- Aperture area [pixels^2]
FLUX_APERTURE 	        	- Total flux w/in aperture [Jy Hz]
ERROR_APERTURE	 		- Uncertainty in flux w/in annulus [Jy Hz]
APERTURE_TOTAL 	        	- Sum of pixel HI surface densities w/in aperture 
				  [Msol * pixel^2 / pc^2]
SURFACE_DENSITY_APERTURE	- Average HI surface density w/in aperture [Msol / pc^2]
