Description of *galex_photometry.fits files.

####### *galex_photometry.fits #######

Columns 1-37            - Default SoFiA catalogue columns

RADIUS_NUV	 	- NUV-band 28 mag /arcsec^2 isophotal radius [arcsec]
MAG_NUV 	        - NUV-band asymptotic magnitude [mag]
ERROR_NUV 	        - NUV-band magnitude uncertainty (standard deviation of 
			  derivative of curve of growth: Delta mag < 0.05) [mag]
SNR_NUV 	        - NUV-band signal to noise ratio (ratio of ADU flux to 
			  uncertainty in ADU flux w/in RADIUS_NUV aperture)
